Typeform uses a long-lived API key. Authsome stores the key encrypted in the local vault, injects it at request time through the proxy, and keeps it out of shell history, process listings, and environment dumps.

## At a glance

| | |
|---|---|
| Provider name | `typeform` |
| Display name | Typeform |
| Auth type | API key |
| Header | `Authorization: Bearer <key>` |
| Proxy host | `api.typeform.com` |
| Env var | `TYPEFORM_API_KEY` |
| Where to get a key | [https://admin.typeform.com/account#/section/tokens](https://admin.typeform.com/account#/section/tokens) |

## Get a key

Create an API key in the Typeform dashboard at [https://admin.typeform.com/account#/section/tokens](https://admin.typeform.com/account#/section/tokens).

## Log in

```bash
uvx authsome login typeform
```

A local browser form opens at `http://127.0.0.1:7998`. Paste the key into the masked input and submit.

<MaskedInputNote />

Verify:

```bash
uvx authsome get typeform --field status
# → connected
```

## Use the key

<CodeGroup>

```python Python library
from authsome.server.dependencies import create_auth_service

auth = create_auth_service()
key = auth.get_access_token("typeform", connection="default")
```

```bash Proxy
uvx authsome run -- python my_agent.py
```

```bash Environment
eval "$(uvx authsome export typeform --format env)"
echo $TYPEFORM_API_KEY
```

</CodeGroup>

Under the proxy, authsome sets `TYPEFORM_API_KEY=authsome-proxy-managed` in the child's environment and injects the real key into outbound requests to `api.typeform.com`. The child process never sees the actual value.

## Multiple keys

<MultiConnectionsCta />

```bash
uvx authsome login typeform --connection personal
uvx authsome login typeform --connection team
```

## Rotate or remove the key

```bash
uvx authsome login typeform --force          # rotate
uvx authsome logout typeform                  # remove local credential
uvx authsome remove typeform                  # remove all local state for the provider
```

API-key providers have no revocation endpoint, so `revoke` and `remove` are equivalent for Typeform.

## Override the bundled definition

```bash
uvx authsome inspect typeform > ~/.authsome/providers/typeform.json
# edit fields
uvx authsome list   # source now shows "custom" for typeform
```

See [Provider schema](/reference/provider-schema) for every field.
