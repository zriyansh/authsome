Rewardful uses a long-lived API key. Authsome stores the key encrypted in the local vault, injects it at request time through the proxy, and keeps it out of shell history, process listings, and environment dumps.

## At a glance

| | |
|---|---|
| Provider name | `rewardful` |
| Display name | Rewardful |
| Auth type | API key |
| Header | `Authorization: Bearer <key>` |
| Proxy host | `api.getrewardful.com` |
| Env var | `REWARDFUL_API_KEY` |
| Where to get a key | [https://app.getrewardful.com/company/api-keys](https://app.getrewardful.com/company/api-keys) |

## Get a key

Create an API key in the Rewardful dashboard at [https://app.getrewardful.com/company/api-keys](https://app.getrewardful.com/company/api-keys).

## Log in

```bash
uvx authsome login rewardful
```

A local browser form opens at `http://127.0.0.1:7999`. Paste the key into the masked input and submit.

<MaskedInputNote />

Verify:

```bash
uvx authsome get rewardful --field status
# → connected
```

## Use the key

<ProxyInjection envVar="REWARDFUL_API_KEY" secretLabel="key" host="api.getrewardful.com" />

### From a Python agent

```python
from authsome.context import AuthsomeContext

ctx = AuthsomeContext.create()
key = ctx.auth.get_access_token("rewardful", connection="default")
```

### As an environment variable

```bash
eval "$(uvx authsome export rewardful --format env)"
echo $REWARDFUL_API_KEY
```

## Multiple keys

<MultiConnectionsCta />

```bash
uvx authsome login rewardful --connection personal
uvx authsome login rewardful --connection team
```

## Rotate or remove the key

```bash
uvx authsome login rewardful --force          # rotate
uvx authsome logout rewardful                  # remove local credential
uvx authsome remove rewardful                  # remove all local state for the provider
```

API-key providers have no revocation endpoint, so `revoke` and `remove` are equivalent for Rewardful.

## Override the bundled definition

```bash
uvx authsome inspect rewardful > ~/.authsome/providers/rewardful.json
# edit fields
uvx authsome list   # source now shows "custom" for rewardful
```

See [Provider schema](/reference/provider-schema) for every field.
