Ashby uses a long-lived API key. Authsome stores the key encrypted in the local vault, injects it at request time through the proxy, and keeps it out of shell history, process listings, and environment dumps.

## At a glance

| | |
|---|---|
| Provider name | `ashby` |
| Display name | Ashby |
| Auth type | API key |
| Header | `Authorization: Basic <key>` |
| Proxy host | `api.ashbyhq.com` |
| Env var | `ASHBY_API_KEY` |
| Where to get a key | [https://app.ashbyhq.com/admin/api-keys](https://app.ashbyhq.com/admin/api-keys) |

## Get a key

Create an API key in the Ashby dashboard at [https://app.ashbyhq.com/admin/api-keys](https://app.ashbyhq.com/admin/api-keys).

## Log in

```bash
uvx authsome login ashby
```

A local browser form opens at `http://127.0.0.1:7999`. Paste the key into the masked input and submit.

<MaskedInputNote />

Verify:

```bash
uvx authsome get ashby --field status
# → connected
```

## Use the key

<ProxyInjection envVar="ASHBY_API_KEY" secretLabel="key" host="api.ashbyhq.com" />

### From a Python agent

```python
from authsome.context import AuthsomeContext

ctx = AuthsomeContext.create()
key = ctx.auth.get_access_token("ashby", connection="default")
```

### As an environment variable

```bash
eval "$(uvx authsome export ashby --format env)"
echo $ASHBY_API_KEY
```

## Multiple keys

<MultiConnectionsCta />

```bash
uvx authsome login ashby --connection personal
uvx authsome login ashby --connection team
```

## Rotate or remove the key

```bash
uvx authsome login ashby --force          # rotate
uvx authsome logout ashby                  # remove local credential
uvx authsome remove ashby                  # remove all local state for the provider
```

API-key providers have no revocation endpoint, so `revoke` and `remove` are equivalent for Ashby.

## Override the bundled definition

```bash
uvx authsome inspect ashby > ~/.authsome/providers/ashby.json
# edit fields
uvx authsome list   # source now shows "custom" for ashby
```

See [Provider schema](/reference/provider-schema) for every field.
