Tolt uses a long-lived API key. Authsome stores the key encrypted in the local vault, injects it at request time through the proxy, and keeps it out of shell history, process listings, and environment dumps.

## At a glance

| | |
|---|---|
| Provider name | `tolt` |
| Display name | Tolt |
| Auth type | API key |
| Header | `Authorization: Bearer <key>` |
| Proxy host | `api.tolt.com` |
| Env var | `TOLT_API_KEY` |
| Where to get a key | [https://app.tolt.com/settings/api](https://app.tolt.com/settings/api) |

## Get a key

Create an API key in the Tolt dashboard at [https://app.tolt.com/settings/api](https://app.tolt.com/settings/api).

## Log in

```bash
authsome login tolt
```

A local browser form opens at `http://127.0.0.1:7998`. Paste the key into the masked input and submit.

<MaskedInputNote />

Verify:

```bash
authsome get tolt --field status
# → connected
```

## Use the key

Run the agent under the proxy. The library tab is for embedding authsome inside a larger Python orchestrator.

<CodeGroup>

```bash Proxy (recommended)
authsome run -- python my_agent.py
```

```bash Environment
eval "$(authsome export tolt --format env)"
echo $TOLT_API_KEY
```

```python Python library
from authsome.server.dependencies import create_auth_service

auth = create_auth_service()
key = auth.get_access_token("tolt", connection="default")
```

</CodeGroup>

Under the proxy, authsome sets `TOLT_API_KEY=authsome-proxy-managed` in the child's environment and injects the real key into outbound requests to `api.tolt.com`. The child process never sees the actual value.

## Multiple keys

<MultiConnectionsCta />

```bash
authsome login tolt --connection personal
authsome login tolt --connection team
```

## Rotate or remove the key

```bash
authsome login tolt --force          # rotate
authsome logout tolt                  # remove local credential
authsome remove tolt                  # remove all local state for the provider
```

API-key providers have no revocation endpoint, so `revoke` and `remove` are equivalent for Tolt.

## Override the bundled definition

```bash
authsome inspect tolt > ~/.authsome/providers/tolt.json
# edit fields
authsome list   # source now shows "custom" for tolt
```

See [Provider schema](/reference/provider-schema) for every field.
